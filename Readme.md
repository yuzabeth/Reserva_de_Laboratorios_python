# Sistema de Reserva de Laboratorios

Aplicación desarrollada en Python que permite gestionar la reserva de laboratorios dentro de un entorno académico. 
El sistema facilita la administración de usuarios, laboratorios y horarios de reserva mediante una estructura modular del código.

El proyecto está organizado siguiendo una separación de responsabilidades entre las capas del sistema, lo que permite mantener una estructura clara y escalable.

## Funcionalidades

- Registro y gestión de usuarios
- Administración de laboratorios disponibles
- Reserva de laboratorios en horarios específicos
- Consulta de información almacenada
- Persistencia de datos mediante archivos

## Estructura del proyecto

El proyecto se encuentra organizado en diferentes módulos:

- **controlador**: lógica del sistema y gestión de procesos
- **modelo**: definición de entidades y manejo de datos
- **vista**: interacción con el usuario
- **interfaces**: componentes de comunicación entre módulos
- **imagenes**: recursos visuales del sistema

## Tecnologías utilizadas

- Python
- Programación modular
- Manejo de archivos para almacenamiento de datos

## Objetivo

El objetivo del proyecto es desarrollar una aplicación que simule el proceso de reserva de laboratorios, aplicando conceptos de programación como modularidad, organización del código y gestión de información.
